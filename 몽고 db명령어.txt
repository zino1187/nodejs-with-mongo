---------------------------------------------------------------
정보조회
---------------------------------------------------------------
● db 생성 및 선택
	use 디비명

● db 목록 보기 
	db.adminCommand( { listDatabases: 1 } )

● 컬렉션 목록 보기 
	db.getCollectionNames();

● 컬렉션안의 데이터 조회하기
	db.컬렉션이름.find();

● 컬렉션안의 데이터 정렬하여 조회하기
	db.컬렉션이름.find();

---------------------------------------------------------------
DDL
---------------------------------------------------------------
● db생성 
	use 만들db명

● db 삭제하기
	use  db명
	db.dropDatabase();

● 컬렉션 만들기
	db.createCollection("컬렉션명");	

● 컬렉션 삭제하기
	db.컬렉션명.drop();

---------------------------------------------------------------
DML
---------------------------------------------------------------
● insert
	db.컬렉션명.save({name:"청바지", price:500});

● delete
	- 조건으로 지우기 db.컬렉션명.remove({"name":"zino"}); 
	- 모두 지우기 db.컬렉션명.remove();
	
● update
	(1)변수에 담기
		var temp=db.컬렉션명.findOne({name:"zino"});

	(2)temp 출력
		temp

	(3)값수정
		temp.값=데이터;
		db.컬렉션명.save(temp);
